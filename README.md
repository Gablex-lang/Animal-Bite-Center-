# 🐾 Animal Bite Center Management System

## 📌 Description

The **Animal Bite Center Management System** is a web-based application designed to manage and organize patient records, treatment data, and staff operations in an animal bite treatment facility.
It helps streamline data handling, improve efficiency, and provide better monitoring of cases.

---

## 🚀 Features

* 🧾 Patient record management
* 💉 Treatment and vaccination tracking
* 👨‍⚕️ Admin, Staff, and Super Admin dashboards
* 🔐 User authentication (login/logout system)
* 📊 Record viewing and printing
* 🗂 Organized database system

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP / Apache

---

## ⚙️ Installation Guide

### 1. Clone the repository

```bash
git clone https://github.com/YOUR-USERNAME/Animal-Bite-Center-.git
```

### 2. Move project to XAMPP

* Copy the project folder
* Paste inside:

```
C:\xampp\htdocs\
```

### 3. Start XAMPP

* Open XAMPP Control Panel
* Start:

  * Apache
  * MySQL

---

### 4. Setup Database

1. Open **phpMyAdmin**
2. Create a new database
3. Import the file:

```
animal_bite_center_db.sql
```

---

### 5. Run the project

Open browser and go to:

```
http://localhost/animal_bite_center_final/
```

---

## 🔑 User Roles

* **Admin** – Manages overall system
* **Staff** – Handles patient records
* **Super Admin** – Full access control
